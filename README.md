#古代战争

##游戏介绍
“古代战争”是一个2.5D即时战略游戏，使用了帝国时代2的素材，并参考了它的游戏设计和玩法。
该游戏基于YEngine2D引擎开发，具备生产、建造、资源采集、战斗、地图迷雾等RTS核心要素。
